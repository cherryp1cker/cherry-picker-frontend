# Cherry Picker

숙박권, 식당 예약권 등을 거래할 수 있는 중고 플랫폼 서비스

## How to use (Local)

1. `npm install`
2. `config/env/.env` 생성하고, 아래 환경 변수 설정
    ```conf
    NCP_CLIENT_ID=<Naver Cloud Platform에서 발급한 Naver Maps Client ID>
    ```
3. `npm run dev`
